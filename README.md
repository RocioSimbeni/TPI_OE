# TPI Organización Empresarial

# Sistema de Gestión de Licencias Laborales Asistido por Chatbot 

## Descripción del Proyecto

El sistema automatiza el proceso tradicional y manual de solicitud de vacaciones de una organización (Proceso AS-IS) , transformándolo en una solución de autogestión disponible las 24 horas a través de un chatbot de Telegram (Proceso TO-BE).  Los empleados pueden consultar su saldo de días disponibles y realizar solicitudes en tiempo real de forma rápida y eficiente.
